Создать закладку для мониторинга

Эта закладка позволяет активировать мониторинг на любом сайте, который вы посещаете. Для использования перетащите ссылку ниже на панель закладок вашего браузера.

Важно:

После перезагрузки страницы клиент автоматически переподключится, если был активирован. Данная версия включает улучшенный механизм автозагрузки.

Google

Инструкция по использованию:

  1. Перетащите ссылку "Google" на панель закладок вашего браузера.
  2. Перейдите на сайт, который нужно мониторить.
  3. Нажмите на закладку "Google" в панели закладок.
  4. Скрипт мониторинга будет активирован и подключится к серверу.
  5. В админ-панели вы увидите новое подключение.
  6. После перезагрузки страницы клиент автоматически переподключится.
